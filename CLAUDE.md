# CLAUDE.md

Claude Code compatibility bootstrap for this repository.

`AGENTS.md` is the single source of truth for repository agent instructions. Read it first, then follow any scoped `AGENTS.md` in the subtree you touch.

Do not duplicate project rules in this file. If an instruction should apply to agents working on Pythinker, update `AGENTS.md` instead.

`CLAUDE.local.md` is gitignored and may contain personal machine/session notes only; it must not override durable project policy unless the user explicitly says so in the current task.
